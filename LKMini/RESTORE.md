# LK Mini Restore

## Minimal Restore Loop

1. Verify SHA256
2. Read Manifest
3. Resolve Locator
4. Load Snapshot
5. Trace ReverseChain
6. Rebuild Package

## Core Rule

A = A

Do not treat:
- generated screenshots
- simulated runtime
- unverifiable packages
- lineage-broken artifacts
as original world objects.

## Public Boundary

LK Mini is a public seed only.
Private runtime systems and personal records are not included.
