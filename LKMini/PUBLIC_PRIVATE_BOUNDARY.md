# Public / Private Boundary

## Public

This repository may publish:

- LK Mini public seed
- README
- NOTICE
- LICENSE
- MANIFEST
- LOCATOR
- SNAPSHOT
- ReverseChain
- RESTORE instructions
- SECURITY policy
- CONTRIBUTING rules
- VERSION history
- public proof anchors
- templates

## Private

This repository must not publish:

- eternal core bodies
- private engine bodies
- private dashboards
- private archives
- credentials
- API keys
- personal records
- family data
- private manifests
- private snapshots
- private reversechains

## Core Rule

A_EQUALS_A=true

Public seed is public seed.
Private engine fleet is private engine fleet.
Hash anchor is hash anchor.
Do not confuse them.
