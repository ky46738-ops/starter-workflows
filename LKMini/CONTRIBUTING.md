## Contribution Boundary

Do not:
- present incomplete runtime as complete
- present generated artifacts as original world objects
- break lineage
- remove verification state
- hide restore failures

## Required Metadata

All contributed artifacts should include:
- source
- lineage
- SHA256 identity
- verification state
- restore state
